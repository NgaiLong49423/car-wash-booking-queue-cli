## Naming Convention

Use a descriptive, stable file name:

```text
Report_<number>_AutoWash_Priority_Booking_Engine.<extension>
```

Store only finalized or reviewable report artifacts here. Temporary exports, screenshots that are not cited by a report, and local office-lock files must not be committed.

## Report Quality Checklist

- Identify the report number, author/team, course context, and date.
- Link claims to the current SRS where they describe product behavior.
- Keep diagrams, benchmark results, and source-code references current.
- Use English for maintained documentation and presentation material.
- Preserve superseded Vietnamese Markdown documentation in [`../old/`](../old/), not in the active report index.
